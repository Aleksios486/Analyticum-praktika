Функционал - регистрация, авторизация пользователя, получение списка постов, конкретного поста любым пользователем, создание, редактирование, удаление поста авторизованным пользователем, автором поста, Создание, получение, редактирование, удаление комментариев, пагинация, FormRequest валидация, использование Resources, комментарии.


Для разворачивания, запуска необходимо перенести в папку home папку blog-api. Нужен OpenServer версии 6.3.5, в нем модули PHP-8.4, MySQL-8.4. Папку vue-blog открыть через консоль, запуск либо через команду npm run dev либо через интерфейс vue ui. Так же необходимо запустить проект blog-api через composer и запустить миграции.
